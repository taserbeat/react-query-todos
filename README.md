# react-query-todos

react-query と Redux Toolkit で作る簡単な Todo アプリ

# 実行方法

1. バックエンドのサーバーをローカルで起動する

詳細は[drf_restapi](https://github.com/taserbeat/drf_restapi)を参考

2. 環境変数を設定

| 環境変数名         | 値                      | 説明                   |
| ------------------ | ----------------------- | ---------------------- |
| REACT_APP_REST_URL | "http://localhost:8000" | バックエンドのホスト名 |

3. フロントエンドのサーバーをローカルで起動する

```bash
yarn install
yarn start
```
